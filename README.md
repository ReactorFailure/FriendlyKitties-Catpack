# Friendly Kitties catpack
A Prism Launcher catpack that randomly shows cat related memes and pictures of my friend's cat

## Installation
1. Go to the [releases page](https://github.com/ReactorFailure/FriendlyKitties-Catpack/releases) and download the latest zip file

2. Find where you saved the zip file and right click and cut

3. Open up Prism Launcher and click on `settings`

4. Go to the `Appearance` tab and click on `Open folder` next to the `Cat Pack` dropdown

5. When the file explorer opens, right click and paste the zip file

6. Unzip the file

7. Go back to Prism and press the `Reload all` button under the Cat Pack dropdown

8. Click the Cat Pack dropdown and find and click on `Friendly Kitties`. A picture of a cat should appear on the bottom left

9. Press `Ok`

The catpack should randomly show a photo every time you open up the launcher or spam press the cat button
